# Project Explanation

This project was created as a **frontend test assignment** built with **React, TypeScript, and Tailwind CSS**.  
The goal was to demonstrate clean component architecture, state management, and responsive UI design using modern frontend tools.

   - Makes components self-documented through props and types.

2. **Tailwind CSS**
   - Used for rapid UI prototyping and consistent design system.
   - Reduces dependency on external UI libraries.
   - Responsive behavior is achieved using Tailwind‚Äôs breakpoint utilities.

3. **Vite as the Build Tool**

4. **Component-driven architecture**

   - Encourages isolation and reusability.

5. **Page-based Routing**
   - The `pages/` folder defines main application views.
   - Optional navigation or route transitions could be implemented using React Router if required.
